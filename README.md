# Farstarbot

Farstarbot is a simple Telegram bot framework to get started on building your own Telegram bot functionalities.

## Installation

To set up the bot, you need to have Python 3.x and pip installed on your Ubuntu 22 system. Run the following command:

```bash
./setup.sh
```

## Usage

To start your bot, run:

```bash
python bot.py
```

## Future Expansions

- Add more message types and command handlers.
- Implement database functionality for persistent data storage.
- Expand with more advanced features like inline queries and custom keyboards.
